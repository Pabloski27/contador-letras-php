# contador-letras-php
Es un pequeño proyecto de clase en PHP que pide al usuario una frase y una letra, después calcula cuantas veces aparece la letra en la frase.

## Funcionalidades

- Pide una frase al usuario
- Pide una letra al usuario
- Cuenta las veces que aparece la letra en la frase
- Muestra las veces que aparece la letra en la frase que le ha pasado el usuario

## Requisitos Previos

**Antes de ejecutar el proyecto tienes que tener instalado:**

- PHP >= 8.0
- Git
